## Build

```
git clone https://github.com/minmin2104/snake
cd snake
python -m venv .venv
```

Before you install the requirements, activate the venv.

On Windows:
```
.\.venv\Scripts\activate
pip install -r requirements.txt
```

On Unix:
```
source .venv/bin/activate
pip install -r requirements.txt
```
